# Проект: Программа раскрытия личности

## Что это

Личный проект Алексея Тарасенко — онлайн-визитка и серия лендингов для продвижения авторской программы по раскрытию личности. Программа помогает людям найти «точку силы» — пересечение талантов, свойств характера и целей. Привлечение клиентов идёт через Telegram (@aleks33375), без автоматической регистрации, оплаты и CRM.

## Для кого

Люди, которые многое попробовали, но не нашли направления: профессионалы без внутреннего ориентира, предприниматели в поиске ниши, те, кто хочет строить жизнь из своей природы, а не вопреки ей.

## Стиль дизайна

- **Тема:** тёмная, почти чёрная (`#080810`)
- **Палитра:** `--coral: #D97070` · `--green: #57B374` · `--blue: #6688CC` · `--gold: #FFD700`
- **Типографика:** системные шрифты (`-apple-system, BlinkMacSystemFont, Segoe UI`)
- **Эффекты:** градиентный текст, размытые фоновые блобы (CSS `filter: blur`), анимация `fadeUp`
- **Квиз:** 4 шага — имя → таланты → характер → цели → приоритет → результат с архетипом и персонализированным заголовком
- **Стек:** чистый HTML + CSS + JS, никаких фреймворков. `html2canvas` для сохранения результата как фото.
