## Запланировано

### 1. Техническая база — сделать первым
- [ ] Добавить OG-метатеги (`og:title`, `og:description`, `og:image`) — без них ссылка не раскрывается в Telegram и ВКонтакте
- [x] Добавить favicon — вкладка браузера без иконки выглядит незаконченно
- [x] Добавить `font-display: swap` в подключение Google Fonts — шрифт не должен блокировать рендер при медленном интернете

### 2. Производительность
- [x] Добавить `will-change: transform` на `.blob` — три постоянно анимирующихся элемента без подсказки браузеру, возможны лишние перерисовки
- [x] Объединить четыре отдельных `addEventListener('scroll', ...)` в один обработчик — снизить нагрузку при каждом событии скролла

### 3. Доступность
- [x] Добавить `@media (prefers-reduced-motion: reduce)` — отключить анимации для пользователей с вестибулярными расстройствами, это стандарт доступности

### 4. Контент
- [ ] Заменить placeholder в секции «Обо мне» на реальное фото — живое, не студийное, placeholder не создаёт доверие

### 5. Адаптивная вёрстка
- [x] Показывать бургер-меню на планшете (до 900px), не только на мобиле (600px) — при 700px навигация уже сжата
- [x] Исправить смещение вертикального таймлайна на мобиле — `left: 1.2rem` при `padding-left: 1rem` создаёт небольшой сдвиг линии
- [x] Убрать `clip-path: polygon(92%)` из hero — заменён на gradient-fade к следующей секции, больше не зависит от цвета фона ниже
- [x] Исправить позицию кнопки «наверх» при открытой клавиатуре на мобиле — `bottom: 1.5rem` перекрывается нативной клавиатурой при фокусе на поле формы

### 6. Анимации — полировка
- [x] Убрать агрессивное скрытие текста в hero при параллаксе — `opacity` теперь падает до 0.2 (было до 0) при прокрутке 1.5× высоты экрана
- [x] Починить конфликт трансформов на карточках — дублирующее объявление удалено, остался один блок `card:hover` + единый `transition` в base CSS
- [x] Ограничить shimmer на section-label — срабатывает один раз через IntersectionObserver, больше не мерцает постоянно
- [x] Починить счётчик + gradient-text — `-webkit-text-fill-color: transparent` конфликтует с `textContent` при анимации, цифры могут мелькать прозрачным

### 7. Интерактивность — UX
- [x] Добавить текстовые подписи к ошибкам формы — поле подсвечивается красным, но нет сообщения «Заполни это поле», особенно критично на мобиле
