# Brief: Лендинг-портфолио цифрового мастера
> Главный документ проекта. Версия 1.0 — апрель 2026.
> Основан на research.md: анализ 10 конкурентов, структура продающих лендингов, тренды 2025–2026, оценка трёх экспертов.

---

## 1. ЦЕЛЬ И АУДИТОРИЯ

### Цель сайта

Конвертировать посетителя в заявку через Telegram. Без форм, без регистрации, без CRM. Одно целевое действие — написать.

**Вторичные цели:**
- Сформировать доверие до первого контакта (кейсы, цифры, отзывы)
- Объяснить нетехничному клиенту что такое AI-агент через то, что он делает
- Отстроиться от конкурентов блоком «Как работаю» (у большинства его нет)

### Целевая аудитория

**Основной сегмент — владелец малого бизнеса / стартапа:**
- 28–45 лет, Россия / СНГ
- Есть бизнес, нет нормального онлайн-присутствия или автоматизации
- Не разбирается в технологиях, думает результатами («сколько заявок», «сколько времени сэкономлю»)
- Ищет конкретного человека, не агентство
- Сидит в Telegram, принимает решения быстро

**Вторичный сегмент — основатель / product-менеджер:**
- Понимает технологии, оценивает кейсы и стек
- Хочет видеть живые проекты, не описания
- Доверяет GitHub-активности и конкретным числам

**Что оба сегмента хотят знать за первые 5 секунд:**
1. Это для меня?
2. Он умеет то, что мне нужно?
3. Как начать и насколько это просто?

## 4. ДИЗАЙН

### Цветовая палитра

```css
--dark:   #080810;   /* фон страницы */
--dark2:  #0e0e1a;   /* фон карточек */
--coral:  #D97070;   /* акцент 1, CTA */
--green:  #57B374;   /* акцент 2, успех, статус */
--blue:   #6688CC;   /* акцент 3, ссылки */
--gold:   #FFD700;   /* акцент 4, highlights */
--text:   #e8e8f0;   /* основной текст */
--muted:  #8888aa;   /* вторичный текст, подписи */
```

**Градиент текста (заголовки):**
```css
background: linear-gradient(90deg, #D97070, #FFD700, #57B374);
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
background-size: 200%;
animation: gradientShift 6s linear infinite;
```

**CTA-кнопка:**
```css
background: linear-gradient(135deg, #D97070, #6688CC);
animation: pulse-cta 2.5s ease-in-out infinite;
```

**Карточки:**
```css
background: rgba(14, 14, 26, 0.7);
border: 1px solid rgba(255, 255, 255, 0.07);
backdrop-filter: blur(8px);
/* hover: цветная тень по теме карточки */
```

### Типографика

**Основной шрифт:** `Manrope` (variable, Google Fonts) — весовой диапазон 200–800, есть кириллица.

**Альтернатива:** `Inter` или `Plus Jakarta Sans`.

**Размеры (clamp):**
```css
--h1:    clamp(2.4rem, 7vw, 5.5rem);    /* Hero заголовок */
--h2:    clamp(1.8rem, 4vw, 3rem);      /* Секционные заголовки */
--h3:    clamp(1.2rem, 2.5vw, 1.6rem);  /* Карточки */
--body:  clamp(1rem, 1.5vw, 1.125rem);  /* Основной текст */
--small: clamp(0.8rem, 1.2vw, 0.9rem);  /* Подписи, теги */
```

**Вес:** ключевые слова `font-weight: 800`, служебные `font-weight: 300`.

### Настроение

- Тёмная тема, фон `#080810`
- Кинематографическая глубина: блобы → зерно → контент
- Живой, не холодный — цветовые акценты и анимации
- Честный и прямой — минимализм без «wow ради wow»
- Человек, не бренд

### Magnetic Button (только десктоп)

```js
document.querySelectorAll('.btn-magnetic').forEach(btn => {
  btn.addEventListener('mousemove', (e) => {
    const rect = btn.getBoundingClientRect();
    const dx = (e.clientX - rect.left - rect.width / 2) * 0.3;
    const dy = (e.clientY - rect.top - rect.height / 2) * 0.3;
    btn.style.transform = `translate(${dx}px, ${dy}px)`;
  });
  btn.addEventListener('mouseenter', () => btn.style.transition = 'none');
  btn.addEventListener('mouseleave', () => {
    btn.style.transition = 'transform 0.5s cubic-bezier(0.23,1,0.32,1)';
    btn.style.transform = '';
  });
});
```

### Ripple на кнопках

```js
document.querySelectorAll('.btn').forEach(btn => {
  btn.addEventListener('click', (e) => {
    const ripple = document.createElement('span');
    const rect = btn.getBoundingClientRect();
    ripple.style.left = `${e.clientX - rect.left}px`;
    ripple.style.top  = `${e.clientY - rect.top}px`;
    ripple.classList.add('ripple');
    btn.appendChild(ripple);
    ripple.addEventListener('animationend', () => ripple.remove());
  });
});
```

## 6. АДАПТИВНАЯ ВЁРСТКА

### Контрольные точки

```css
@media (max-width: 900px) { /* планшет — двухколоночные сетки */ }
@media (max-width: 600px) { /* мобиль — одна колонка, меньше отступы */ }
```

### Правила

- `clamp()` для шрифтов — не медиазапросы
- `max-width: 1200px` на обёртках, `padding: 0 clamp(1rem, 5vw, 2rem)` по бокам
- Hero grid → стек на мобиле (текст сверху, визуал снизу)
- Bento grid: `repeat(auto-fit, minmax(280px, 1fr))`
- Таймлайн «Как работаю»: горизонт → вертикаль
- Кастомный курсор: `@media (pointer: coarse) { display: none }`
- Magnetic кнопка: отключить на touch-устройствах
- Кнопки: минимум 44×44px на мобиле

```
HTML5 + CSS3 + Vanilla JS
Без фреймворков, без сборщиков

Иконки: SVG инлайн или Lucide
Анимации: CSS keyframes + IntersectionObserver + CSS Scroll Timeline API
Контакт: ссылка t.me/[username] — без форм и полей
```

**Оптимизация:**
- Изображения в WebP + `srcset` для Retina
- Шрифты: `font-display: swap`
- `rel="preconnect"` для Google Fonts
- Критический CSS инлайн в `<head>`
